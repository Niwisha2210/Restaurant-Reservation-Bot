A Python-based restaurant reservation chatbot with a friendly GUI using Tkinter. Smart Dine allows users to view the menu, check table availability, book tables, and see booking history through a chat-like interface.

1. Introduction
Smart Dine helps users book restaurant tables easily without phone calls or web forms.
It provides a friendly conversational interface and real-time seat availability, making reservations faster and more convenient.

2. Problem
Booking a table manually can be slow and inconvenient.
Smart Dine automates the process and provides instant feedback on availability, bookings, and menu options.

3. Objectives
Create a chat-based GUI using Python and Tkinter.
Allow users to check menu and availability.
Enable booking and managing reservations.
Store user booking history.

4. How It Works
User starts the chatbot and enters their name.
The bot shows options like Menu, Availability, Book, Summary, History.
For booking:
User selects date and time.
User selects number of seats.
Bot checks availability and confirms booking or suggests alternatives.
All bookings are stored in memory for summary and history.

5. Features
View Menu – See all dishes with prices.
Check Availability – See free tables for chosen dates.
Book Table – Reserve tables with seat validation.
Booking Summary – See your last booking.
Booking History – Track all past reservations.
Error Handling – Suggest alternative slots if requested seats are not available.

6. Creative Feature
Makes the booking experience interactive and fun, not just text-based.
Users can see which tables are free/occupied.
When a user hovers over a table, show table capacity.

7. Advantages
Easy to use with a chat interface.
Fast and interactive table booking.
Personalized user history and summaries.

8. Future Work
Add database storage for persistent bookings.
Include AI-based dish recommendations.
Implement interactive table layout for visual booking.
Support multi-language chat.
Send booking reminders via email.

9.Conclusion
Smart Dine shows how a chatbot and GUI can simplify restaurant reservations. It combines Python programming, GUI design, and data handling to create a friendly and interactive experience.
